<h4 href="https://k6.io/docs/get-started/running-k6/">Running K6 Scripts</h4>
Example of running K6 load test with 100 users for 30 seconds:

```
k6 run --quiet --vus 100 --duration 30s --iterations 1000000 script.js
```
